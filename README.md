# 清理相同相片，清理相册

清理相同相片项目，基于`Swift`语言，架构为`MVVM`，采用主要第三方库为 `RxSwift` + `Swinject` + `Moya`，导航采用 `Coordinator`模式，采用`opencv`识别



## 安装方法

```shell
git clone & pod install
```
